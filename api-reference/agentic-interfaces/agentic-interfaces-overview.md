Agentic Interfaces allows you to interact with HiPay APIs directly in LLMs.

## MCP server

The HiPay MCP Server acts as a bridge between AI agents and HiPay's Enterprise APIs.

It enables seamless retrieval of information and automated actions within HiPay, making integrations faster, smarter, and more efficient.
