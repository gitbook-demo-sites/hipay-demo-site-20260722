# CMS Modules - Overview

{% hint style="info" %}
Imported from the current HiPay WordPress developer portal for the demo migration. Source: [https://developer.hipay.com/cms-modules/cms-modules-overview](https://developer.hipay.com/cms-modules/cms-modules-overview)
{% endhint %}

In case you use a CMS to handle your website, you can install our HiPay Module for that CMS to simplify the payment integration.

Each HiPay CMS Module will have its own administration section, where you will be able to customize your integration.

Currently, we have five different CMS Modules that you can benefit from :

Magento 2 is a CMS which is a blend of highly scalable design and a wide-ranging support network. More information about how to install the Magento 2 Module here.

Prestashop is an extremely versatile platform and a highly-rated development program. More information about how to install the Prestashop 1.6 - 1.7 Entreprise Module or the Prestashop Professional Module here.

Shopify is a Saas platform with a plugin/application style. Great for medium size merchants as it is easy and cheap to maintain. When you want more custom features, it becomes a bit complex. More information about how to install our HiPay Module here.

Like other eCommerce platforms, Salesforce Commerce Cloud is a cloud-based software used by businesses to sell products online. More information about how to install the SFCC Module here.

WSHOP is the publisher of a Digital Unified Commerce Solution. It provides retailers and pure players with a wide set of tools and functionalities to deploy their e-commerce & phygital (Digital In Store). Ask for a demo and or contact your technical account manager for more information.

Proximis' value proposition is the a simple integration for the Unified Commerce. If you want to have HiPay as PSP on your Proximis website, please contact your technical account manager.

WooCommerce inherits all of the strength and reliability of WordPress and uses it to build a world-class eCommerce experience for retailers and their customers.

More information about how to install the Woocommerce Module here.

Wizishop is a French solution for creating e-commerce websites by subscription. This solution has been on the market for many years and runs nearly 10,000 online shops, mainly in France. More information about how to install the WiziShop Entreprise Module or the Professional Module here.
